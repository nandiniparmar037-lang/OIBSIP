# BMI Calculator

## Description
This is a Python project that calculates Body Mass Index (BMI) using a user's weight and height.

## Features
- Takes weight input from the user
- Takes height input from the user
- Calculates BMI
- Displays BMI category:
  - Underweight
  - Normal Weight
  - Overweight
  - Obese

## Technologies Used
- Python

## How to Run
1. Open the project in VS Code.
2. Run the file bmi_calculator.py.
3. Enter your weight in kilograms.
4. Enter your height in meters.
5. View your BMI and category.
