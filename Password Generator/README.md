# Simple Password Generator

## Description

This is a Python project that generates a random and secure password based on the length entered by the user. The password contains uppercase letters, lowercase letters, numbers, and special characters to make it strong and secure.

## Features

- Generates random passwords
- User can choose the password length
- Includes:
  - Uppercase letters (A-Z)
  - Lowercase letters (a-z)
  - Numbers (0-9)
  - Special symbols (!, @, #, $, %, ^, &, *, ?)
- Simple and easy-to-use command-line interface

## Technologies Used

- Python
- Random Module

## How to Run

1. Open the project in VS Code.
2. Run the `password_generator.py` file.
3. Enter the desired password length.
4. Press Enter.
5. The program will generate and display a secure password.

## Example Output

```
Enter the length of the password: 10

Generated Password: A7@bQ2#mX9
```
